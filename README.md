# 🇮🇳 RashtraSuraksha AI  
## National Traffic Accident Intelligence & Forecasting System

An AI-powered national-level traffic accident analytics platform designed to monitor, predict, and visualize accident risk across India.

---

## 🚀 Key Features

- 🛰 Interactive India Risk Heatmap
- 📊 Real-time Command Center Dashboard
- 🤖 Machine Learning Risk Prediction (Random Forest)
- 📈 Time-Series Forecasting
- 🧠 Explainable AI using SHAP
- 📄 Auto-Generated PDF Intelligence Reports
- 🔐 Admin Data Upload & Model Retraining

---

## 🧠 Machine Learning Pipeline

- Data Cleaning & Feature Engineering
- Label Encoding
- Random Forest Classifier
- Model Evaluation
- SHAP Explainability

---

## 🛰 India Risk Intelligence Map

State-wise accident intensity visualized using GeoJSON and Plotly.

---

## 📊 Forecasting

Time-series analysis to predict future accident trends.

---

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Plotly
- SHAP
- Pandas
- GeoJSON
